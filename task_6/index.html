Задание 6. Бинарный поиск

6.1 Приближенный поиск

Дан упорядоченный по возрастанию массив чисел. Дан второй массив чисел — запросы. Для каждого запроса необходимо вывести число из первого массива наиболее близкое к запрашиваемому.

Входные данные

В первой строке целое число nnn — количество элементов в массиве. Во второй строке nnn целых чисел — элементов массива. В третьей строке число mmm — количество запросов. В четвертой строке mmm целых числе — запросы.

Выходные данные

На выходе mmm чисел через пробел — результаты выполнения запросов.

Пример

Вход

8
1 3 5 8 10 11 14 20
2
19 9

Выход

20 8

6.2 Дипломы

Дано NNN одинаковых дипломов высотой hhh и шириной www. Нужно найти сторону MMM минимально возможного квадратного ящика, в который можно будет разложить все дипломы.

Дипломы нельзя вращать и складывать друг на друга.

Алгоритм должен использовать бинарный поиск.

Входные данные

На входе 3 целых числа: n w h, где 1≤n,w,h≤1091 \le n,w,h \le 10^91≤n,w,h≤109

Выходные данные

На выходе одно целое число — длина стороны минимального квадратного ящика, в который можно расположить все дипломы.

Пример 0

Вход

1 1 1

Выход

1

Пример 1

Вход

10 2 3

Выход

9

Пример 2

Вход

5 1 1

Выход

3

Алгоритм должен найти корни уравнения методом динарного поиска для последовательного приближения, до тех пор, пока не будет достигнута требуемая точность.

Входные данные

Вещественное число 1.0≤a≤10101.0 \le a \le 10^101.0≤a≤1010

Выходные данные

Пример 1

Вход

Выход
